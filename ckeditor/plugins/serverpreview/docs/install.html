Server Preview Plugin for CKEditor

Introduction

This is a plugin that enables to perform a more real preview of the contents of CKEditor.

No new dialogs, commands or toolbar buttons are added, the default Preview button is modified instead.

Installation

1. Copying the files

Extract the contents of the zip in you plugins directory, so it ends up like this

ckeditor\
	...
	images\
	lang\
	plugins\
		...
		serverpreview\
			plugin.js
			docs\
		...
	skins\
	themes\

2. Adding it to CKEditor

Now add the plugin in your config.js or custom js configuration file: config.extraPlugins='serverpreview'; You'll also need to specify the path to the server script that will handle the posted data: config.serverPreview_Url='/path/page.php'; The data is sent by default as a form element named 'htmlData' and can be changed this way: config.serverPreview_FieldName = 'content-preview'; It's possible to specify a set of fields that should be copied to the preview form by adding them to config.serverPreview_AdditionalFields separated by semicolons: config.serverPreview_AdditionalFields='type;id'; of course, you can change instead the serverPreview_Url if you prefer to get the fields as GET: config.serverPreview_Url='/path/page.php?hidden-pid=' + document.getElementById('hidden-pid').value;

3. Use it

Now empty the cache of your browser and reload the editor, the preview button now will launch the page in your server.
